README - PLACE IA Web Files

Arquivos incluídos:
- index.html: Página inicial com menu e logo
- login.html: Página de login para clientes
- style.css: Estilos principais com tema neon azul e preto escuro
- login.css: Estilos específicos para a página de login
- README.txt: Este arquivo com instruções

Como usar:
1. Coloque todos os arquivos na mesma pasta.
2. Abra o arquivo index.html no navegador para acessar a página inicial.
3. Use o link "Entrar" para acessar a página de login (login.html).
4. Verifique se o navegador tem acesso à internet para carregar os ícones Font Awesome.
5. Para personalizar, edite os arquivos CSS conforme desejar.
